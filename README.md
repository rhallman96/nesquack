# nesquack
An NES emulator written in Golang. Quack quack!

## Controls
### NES Gamepad 1
* arrow keys - joypad
* x - A button
* z - B button
* right shift - select
* return - start

## Supported Mappers
* [NROM](https://wiki.nesdev.com/w/index.php/NROM)
* [MMC1](https://wiki.nesdev.com/w/index.php/MMC1)
